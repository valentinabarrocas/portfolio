# Valentina Barrocas — Portfolio

Portfólio pessoal focado na intersecção entre design editorial, arte interativa e desenvolvimento front-end. O projeto foi construído com uma estética minimalista, priorizando a legibilidade e a exposição visual dos projetos como em uma galeria.

## Tecnologias
- HTML5
- CSS3 (Custom Properties & Media Queries)
- JavaScript (Vanilla)
- Google Fonts (Inter)

## Dark Mode
O site conta com implementação de **Dark Mode automático** via CSS, respeitando as preferências de tema do sistema operacional do usuário (`prefers-color-scheme`). 

- **Cores Light:** Fundo branco com textos em tons de cinza grafite.
- **Cores Dark:** Fundo "Off-Black" (#111) com destaques em Rosa (#e6a5cf).

## Responsividade
Layout totalmente responsivo com ajustes específicos para dispositivos mobile, garantindo que a experiência de navegação seja fluida tanto em desktops quanto em telas menores.
