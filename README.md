# valentinabarrocas - portfólio

portfólio pessoal focado na intersecção entre desenvolvimento front-end, design de interface e creative coding. o projeto foi construído com uma estética minimalista, priorizando a legibilidade e a exposição visual dos projetos como em um museu de arte. a identidade visual fica por conta da cor de fundo e detalhes em rosa.

## tecnologias
- html
- css
- javascript

## dark mode
o site conta com implementação de **dark mode automático** via css, respeitando as preferências de tema do sistema operacional do usuário.

- **cores light (anteriormente):** fundo branco com textos em tons de cinza grafite e destaques em rosa.
- **cores light (atualmente):** fundo rosa com textos em preto e destaques em cinza grafite.
- **cores dark:** fundo "off-black" com textos e destaques em rosa.

## responsividade
layout responsivo com ajustes específicos para dispositivos mobile, garantindo que a experiência de navegação seja fluida tanto em desktops quanto em telas menores.
